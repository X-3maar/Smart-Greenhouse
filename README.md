# Smart-Greenhouse (15 hours)
Smart greenhouse is an agricultural project, the idea is based on creating an automated greenhouse system works with sensors and actuators. The project monitors soil moisture, light intensity, humidity, PH and temperature, and continuously checks for these parameters if they're in the perfect condition of the planted crop, if not, the actuators get activated.
